# Atlas OS – Enterprise AI Automation Platform

A premium, responsive SaaS landing page built for the **Next-Gen AI Platform Speed Run Hackathon**. Atlas OS is designed as a modern enterprise automation platform that showcases clean architecture, responsive UI, high-performance frontend engineering, and polished user experience.

## 🚀 Overview

Atlas OS is a production-inspired landing page for an AI-powered business automation platform. The project focuses on engineering quality, responsiveness, accessibility, semantic HTML, and optimized state management while delivering a premium user experience.

---

## ✨ Features

* Premium enterprise SaaS landing page
* Responsive design (Desktop, Tablet & Mobile)
* Interactive Hero section
* Responsive Bento Grid → Accordion transformation
* Matrix-driven multi-currency pricing engine
* Monthly / Annual billing toggle
* Dynamic pricing calculations
* ROI Calculator
* Animated statistics
* Testimonials
* FAQ section
* Modern Call-To-Action
* Responsive Footer
* Semantic HTML & SEO optimization
* Accessible UI components
* Optimized animations using native CSS

## 🚀 Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Production Build

```bash
npm run build
```

### Start Production Server

```bash
npm start
```
